## Development

This project is build on [VuePress](https://v1.vuepress.vuejs.org/) which is a Vue-powered Static Site Generator.

### Steps

- Clone this repository
- `yarn install`
- `yarn docs:dev`

Build the website
- `yarn docs:build`

## Deployment
